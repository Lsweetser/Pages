Application Overview

This website will allow end users to view my professional experience, education, skills, and projects in a clear and organized manner. Users will also be able to contact me directly via a form on the site.

Technology Overview

Languages: HTML, CSS, JavaScript
Libraries/Frameworks: Bootstrap (optional for advanced layouts)
Platform: GitHub Pages
Hosting: Static hosting via GitHub Pages
